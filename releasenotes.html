3.5.1-SNAPSHOT Release Notes

A complete list of issues can be seen in the tracker milestone 3.5.1-SNAPSHOT .

This release features Bourbon version 4.3.4 and Sass version 3.5.3 running on JRuby 9.1.13.0. Please check the full dependencies to find out about the specific Sass and other transitive dependencies included in this release.

New

  • Drop Compass support (#77)

Enhancements and Fixes

  • Update Sass to version 3.5.3, please check the Sass changelog
  • Update SCSS-Lint to version 0.56.0, please check the SCSS-Lint changelog
  • Update Bourbon to version 4.3.4, please check the Bourbon changelog
  • Update JRuby to 9.1.13.0
