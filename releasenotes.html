3.5.6-SNAPSHOT Release Notes

A complete list of issues can be seen in the tracker milestone 3.5.6-SNAPSHOT .

This release features Bourbon version 5.1.0 and Sass version 3.6.0 running on JRuby 9.2.0.0. Please check the full dependencies to find out about the specific Sass and other transitive dependencies included in this release.

Enhancements and Fixes

  • Update SCSS-Lint to version 0.57.1, please check the SCSS-Lint changelog
  • Update Sass to version 3.6.0, please check the Sass changelog Note that Ruby Sass has now been deprecated, read the blogpost:Ruby Sass is Deprecated.
  • Mark plugin threadSafe, fixes (#209)
